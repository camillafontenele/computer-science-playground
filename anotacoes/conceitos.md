# Conceitos

## Algoritmo

Um algoritmo é uma sequência de passos para resolver um problema.

## Entrada

É o dado que o programa recebe.

## Processamento

É o que o programa faz com o dado.

## Saída

É a resposta que o programa devolve.

## Busca linear

Procura item por item.

Complexidade: O(n)

## Busca binária

Procura dividindo a lista pela metade.

Complexidade: O(log n)

Só funciona com listas ordenadas.
