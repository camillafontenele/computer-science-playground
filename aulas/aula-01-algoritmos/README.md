# Aula 01 — Algoritmos e Pensamento Computacional

## Objetivo da aula

Entender o que é um algoritmo, como funciona entrada, processamento e saída, e comparar busca linear com busca binária.

## Conceitos estudados

- Algoritmo
- Entrada
- Processamento
- Saída
- Busca linear
- Busca binária
- Introdução ao Big O

## O que aprendi

Um algoritmo é uma sequência de passos para resolver um problema.

A busca linear percorre item por item até encontrar o valor procurado.

A busca binária divide a lista pela metade a cada tentativa, mas só funciona se a lista estiver ordenada.

## Exercícios feitos

- Buscar nome em uma lista usando busca linear.
- Buscar número em uma lista usando busca binária.
- Comparar busca linear e busca binária.

## Resultado importante

Para encontrar o número 873 entre 1 e 1000:

- Busca linear: 873 tentativas
- Busca binária: 9 tentativas

## Conclusão

A busca binária é muito mais eficiente em listas grandes ordenadas porque reduz o problema pela metade a cada tentativa.
